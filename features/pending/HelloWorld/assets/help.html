Hello World Feature Documentation

Overview

The Hello World feature demonstrates core functionality of the Cobra AI Features plugin, including:

  • Shortcode usage
  • REST API endpoints
  • Database integration
  • Settings management

Shortcode Usage

[helloworld]

This will display your configured hello message and selected choice.

REST API

Endpoint: /wp-json/cobra-ai/v1/hello-world

Method: GET

Example Response:

{
    "message": "Hello World!",
    "choice": "option1"
}

Database

The feature creates a table named wp_cobra_hello_table with the following structure:

  • id (bigint) - Auto-incrementing ID
  • message (varchar) - Hello message
  • choice (enum) - Selected option
  • created_at (datetime) - Creation timestamp

Settings

Configure the feature in the WordPress admin under Cobra AI â†’ Hello World:

  • Hello Message: Customize the greeting message
  • Choose Option: Select from three available options

Example Implementation

Add to your template or post:

<?php echo do_shortcode('[helloworld]'); ?>

API Usage Example

fetch('/wp-json/cobra-ai/v1/hello-world')
    .then(response => response.json())
    .then(data => console.log(data));
