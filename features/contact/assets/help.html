Contact Form Feature Documentation

Getting Started

The Contact Form feature allows you to add customizable contact forms to your website. Visitors can submit messages through these forms, and you can manage and respond to submissions from your WordPress admin panel.

Quick Start: Use the shortcode [cobra_contact_form] to display the contact form on any page or post.

Adding a Contact Form to Your Site

Using the Shortcode

Add the shortcode [cobra_contact_form] to any page, post, or widget area where you want the form to appear.
Customize the form using the available parameters (see "Shortcode Parameters" section below).

Basic Example

[cobra_contact_form title="Get in Touch"]

Advanced Example

[cobra_contact_form 
    title="Contact Our Support Team" 
    subject_options="Technical Support,Feature Request,Bug Report" 
    submit_button_text="Send Support Request"
    success_message="Thank you for contacting our support team. We'll get back to you shortly."
]

Shortcode Parameters

You can customize the contact form using the following parameters:

Basic Parameters

  • title - Adds a heading above the form
  • subject_options - Comma-separated list of subject options for the dropdown
  • show_subject - Set to "yes" or "no" to show or hide the subject field
  • submit_button_text - Custom text for the submit button

Message Parameters

  • success_message - Message displayed after successful submission
  • error_message - Message displayed when an error occurs
  • redirect_url - URL to redirect to after successful submission

Security Parameters

  • use_recaptcha - Set to "yes" or "no" to enable or disable reCAPTCHA for this form

Managing Submissions

You can view and manage all form submissions in the WordPress admin area.

Viewing Submissions

Go to Cobra AI > Contact Submissions in your WordPress admin menu.
You'll see a list of all submissions with their status (unread, read, replied).
Click on a submission to view its details and respond to it.

Replying to Submissions

Open a submission by clicking on its name in the submissions list.
Scroll down to the "Send Response" section.
Enter your response in the text area.
Click the "Send Response" button to send an email reply to the contact.

The contact will receive your response via email. The submission will be marked as "Replied" after your response is sent.

Configuring the Contact Form

You can customize various aspects of the contact form through the settings page.

Accessing Settings

Go to Cobra AI > Contact Form in your WordPress admin menu.
Navigate through the tabs to configure different aspects of the form.

Available Settings

  • Form Fields - Control which fields are displayed and their labels
  • reCAPTCHA - Configure Google reCAPTCHA to prevent spam
  • Notifications - Set up email notifications for new submissions
  • Messages - Customize the various messages displayed to users
  • Styling - Adjust the appearance of the form

Setting Up reCAPTCHA

To protect your form from spam, you can enable Google reCAPTCHA:

Create a reCAPTCHA site key and secret key at https://www.google.com/recaptcha/admin
Go to the reCAPTCHA tab in your Contact Form settings
Enable reCAPTCHA and enter your site key and secret key
Save your settings

Make sure to keep your secret key confidential and never expose it in client-side code.

Email Templates

You can customize both the notification emails sent to administrators and the reply emails sent to contacts.

Notification Email Variables

  • {{site_name}} - Your website's name
  • {{name}} - Sender's name
  • {{email}} - Sender's email
  • {{subject}} - Message subject
  • {{message}} - Message content
  • {{submission_link}} - Link to view the submission in admin

Reply Email Variables

  • {{name}} - Recipient's name
  • {{subject}} - Original message subject
  • {{message}} - Original message content
  • {{response}} - Your response content

Troubleshooting

Form Not Displaying

  • Ensure the shortcode is added correctly with square brackets: [cobra_contact_form]
  • Check if the feature is enabled in Cobra AI Features
  • Try disabling other plugins that might conflict with the form

Form Submissions Not Working

  • Check your browser console for JavaScript errors
  • Ensure your server allows AJAX requests
  • Temporarily disable reCAPTCHA to see if that's causing issues

Email Notifications Not Received

  • Check your spam folder
  • Verify the notification recipients are set correctly
  • Try using an SMTP plugin to improve email delivery
