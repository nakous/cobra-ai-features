FAQ Feature Documentation

Getting Started

The FAQ feature allows you to create and manage a comprehensive FAQ section on your website. You can organize FAQs into categories, enable search functionality, and customize the display options.

Quick Start: Use the shortcode [cobra_faq] to display your FAQs on any page or post.

Creating FAQs

Adding a New FAQ

Navigate to FAQ > Add New in your WordPress dashboard.
Enter the question in the title field.
Add the answer in the main content editor.
Assign categories if needed.
Click "Publish" to make the FAQ live.

Organizing FAQs

Categories

You can organize your FAQs into categories for better organization and filtering:

Go to FAQ > Categories
Add new categories or edit existing ones
Assign FAQs to categories when creating or editing them

Displaying FAQs

Basic Shortcode

[cobra_faq]

Advanced Shortcode Options

[cobra_faq category="general" limit="5" orderby="views" order="DESC"]

Available parameters:

  • category - Display FAQs from a specific category
  • limit - Number of FAQs to display
  • orderby - Sort by: date, title, or views
  • order - Sort order: DESC or ASC

Customizing Display

Settings Options

Navigate to Cobra AI > FAQ Settings to customize:

  • FAQ title text
  • Number of FAQs per page
  • Category display options
  • Sorting preferences
  • Footer text and link
  • Pagination style (pagination or load more)

Search Functionality

Configuration

In the settings, you can:

  • Enable/disable search functionality
  • Enable/disable autocomplete
  • Set minimum characters for search
  • Show/hide category filter in search

Styling Options

Available Themes

  • Default - Standard clean layout
  • Minimal - Simplified design with less visual elements
  • Boxed - Each FAQ in a distinct box with shadow

Additional Style Options

  • Enable/disable animations
  • Enable/disable search result highlighting

Tracking and Analytics

The FAQ feature automatically tracks:

  • View counts for each FAQ
  • Most viewed FAQs
  • User feedback (helpful/not helpful)

Best Practices

Writing Effective FAQs

  • Keep questions clear and concise
  • Group related questions into categories
  • Use plain language that your users understand
  • Regular update your FAQs based on user feedback

Common Issues

  • Ensure your theme's styling doesn't conflict with FAQ styles
  • Test pagination with large numbers of FAQs
  • Verify search functionality works with your content
