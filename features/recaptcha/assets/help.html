Getting Started

Obtaining API Keys

  1. Go to the Google reCAPTCHA Admin Console
  2. Sign in with your Google account
  3. Click the "+" or "Create" button
  4. Enter a label for your site (e.g., "My WordPress Site")
  5. Select the reCAPTCHA type you want to use
  6. Add your domain(s) to the allowed domains list
  7. Accept the Terms of Service
  8. Click "Submit"
  9. Copy your Site Key and Secret Key

Important: Never share your Secret Key publicly. The Site Key is meant to be public, but the Secret Key must be kept confidential.

reCAPTCHA Versions

Version Description Best For
reCAPTCHA v2 Checkbox Traditional checkbox-based verification with optional challenge Most common forms where user interaction is expected
reCAPTCHA v3 Invisible score-based verification without user interaction High-traffic sites needing frictionless verification
Invisible reCAPTCHA No checkbox, shows challenge only when necessary Forms where minimal user interaction is preferred

Configuration

Basic Settings

Setting Description
Version Select which reCAPTCHA version to use
Theme Choose between Light and Dark themes (v2 only)
Badge Position Where to display the reCAPTCHA badge (v3 and Invisible)
Score Threshold Minimum score to pass verification (v3 only, 0.0 to 1.0)

Custom Form Integration

To protect a custom form, you can either:

  1. Enable "Custom Forms" in the settings and add the form's CSS selector
  2. Use the following shortcode in your form:
[recaptcha]

Troubleshooting

Common Issues

reCAPTCHA Not Showing

  1. Verify your Site Key is correct
  2. Check if your domain is allowed in reCAPTCHA settings
  3. Ensure JavaScript is enabled and loading properly
  4. Check browser console for errors

Verification Failing

  1. Verify your Secret Key is correct
  2. Check if your server can reach Google's servers
  3. Ensure your server time is synchronized
  4. Check server logs for specific error messages
