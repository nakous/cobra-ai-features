Stripe Integration Feature

Overview

The Stripe feature provides core integration with Stripe's API, including:

  • API Key Management
  • Webhook Processing
  • Event Handling
  • Logging System

Configuration

  1. Enter your Stripe API keys (test and live)
  2. Configure your webhook endpoint: /wp-json/cobra-ai/v1/stripe/webhook
  3. Add the webhook secret to your settings

Using in Other Features

Listen for Stripe events in your features:


    add_action('cobra_ai_stripe_payment_intent_succeeded', function($event) {
        // Handle successful payment
    });

API Access

Get API instance in your features:


    $stripe = cobra_ai()->get_feature('stripe');
    $api = $stripe->get_api();
    $secret_key = $api->get_secret_key();
